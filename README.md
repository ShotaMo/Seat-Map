# 座席表作成アプリ

会場内のテーブルと座席を視覚的に配置し、各座席にメンバーを割り当てることができるWebアプリケーションです。

## 特徴

- 直感的なUIで簡単に座席配置を作成
- テーブルを最大6個まで配置可能
- 各テーブルに最大8個の椅子を配置可能
- 各座席にメンバー情報（名前、X ID）を登録可能
- レスポンシブデザイン対応

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - 型安全な開発
- **Vite** - 高速ビルドツール
- **CSS** - スタイリング

## セットアップ方法

### 必要な環境

- Node.js 18以上
- npm または yarn

### インストール

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview
```

## 使い方

### 1. テーブルを追加

1. ヘッダーの「テーブルを追加」ボタンをクリック
2. 会場にテーブルが追加されます（最大6個）

### 2. 椅子を追加

1. テーブル内の「椅子を追加」ボタンをクリック
2. テーブルに椅子が追加されます（1テーブルあたり最大8個）

### 3. メンバー情報を登録

1. 椅子をクリック
2. モーダルが表示されるので、名前とX IDを入力
3. 「保存」ボタンをクリック

### 4. メンバー情報を編集

1. メンバーが登録されている椅子をクリック
2. 情報を編集して「保存」ボタンをクリック

### 5. メンバー情報を削除

1. メンバーが登録されている椅子をクリック
2. 「クリア」ボタンをクリック

### 6. 椅子を削除

1. 椅子にマウスをホバー
2. 右上に表示される「×」ボタンをクリック

### 7. テーブルを削除

1. テーブル右上の「削除」ボタンをクリック

## 開発プロセス

このプロジェクトはSpec駆動開発で作成されました：

1. **Spec.md** - 要件定義
   - 機能要件と非機能要件を明確化
   - 制約事項と画面構成を定義

2. **Design.md** - 設計
   - 技術スタックの選定
   - データモデルとコンポーネント設計
   - UI/UXデザインの詳細化

3. **Tasks.md** - 実装計画
   - 実装フェーズの分割
   - タスクの優先順位付け
   - 完了条件の明確化
